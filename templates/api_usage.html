API Usage Instructions

Upload API

Endpoint: /api/upload

Method: POST

Use this endpoint to upload files. The files should be sent as multipart/form-data.

Download API

Endpoint: /api/download/<link>?user_id=<user_id>

Method: GET

Use this endpoint to download files associated with a specific link. You must provide the user ID as a query parameter. If the user ID does not match the ID associated with the files, or if the ID is not -1 (indicating public files), access will be denied.

Example request: curl -o download.zip "http://localhost:5000/api/download/12345?user_id=67890"
